# inner createUser(body, cb)

Main function used for email sign up. It creates a firebase email user first, then creates a mongo user using the parameter body. Once the user is created in both databases, a verification email is sent. If mongo fails the user is deleted from firebase.

NOTE- if body.uid is present - the function straight away creates a mongo user and skips any firebase api calls as we only need the firebase uid to create a mongo user and "link" it to firebase.

Parameters:
Name Type Description
body userObject
cb function

After all requests are successfull the user object returned from mongo is passed to the callback. Null is passed if anything fails.
