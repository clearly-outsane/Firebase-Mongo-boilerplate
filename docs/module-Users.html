# inner createUser(body, cb)

Function used for email sign up. It creates a firebase email user first. Upon completing that async request, it then creates a mongo user using the Object passed as the body argument. Once the user collection is created in both databases, a verification email is sent. If mongo fails the user collection is deleted from firebase.

NOTE- if body.uid is present - the function straight away creates a mongo user collection and skips any firebase api calls involving the user.

Parameters:
Name Type Description
body userObject
cb function

After all requests are successfull the user object returned from mongo is passed to the callback. Null is passed if anything fails.

# inner emailSignIn(state, dispatch, email, password, getMongoUser)

Function used to sign in user using email and password. Firebase is used for authentication. Once user is signed into firebase, the corresponding mongo user collection is fetched. If mongo user doesn't exist then the user will be null in state. This is later used to show the correct form to the user - to fill in any remaining details after which a mongo user is created.
